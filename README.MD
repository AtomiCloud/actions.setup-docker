# Setup AtomiCloud's Docker Action

Setup AtomiCloud CI Docker (Compatible with Namespace)

# Get Started

Instruction

```yaml
name: Example Workflow
on: push
jobs:
  Test:
    runs-on: ubuntu-latest
    steps:
      # change this
      - uses: AtomiCloud/actions.setup-docker @v1.0.0
```
